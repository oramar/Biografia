Iniciando curso de Git. 23 de Noviembre de 2020

git init 								= 	Inicializa a git
git add . 								= 	Agrega todo para hacer commit
git commit -a							=	Abre vim para dar una descripcion mas exacta al commit
git commit -m "Mensaje Commit"			=	Crear un commit con mensaje rapido
git log --oneline						=	Muestra los commits creados pero con el codigo resumido
git log 								=	Muestra los commits mas detallados
git checkout codigoCommit 				=	Regresa a ese numero de commit
git checkout master						=	va al ultimo commit creado
git show								=	Mustra que cambio mas detallado ha pasado en los commit
git reset codigoCommit --hard			=	Elimina todos los commit creados despues del codigo commit mensionado


Trabajando Con Ramas

git branch								=	Alista todas las ramas creadas que tienes
git branch nombreRama					= 	Crea una nueva rama con el nombre indicado
git checkout nombreRama					=	Ir a otra rama
